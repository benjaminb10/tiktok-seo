# Claude Code Instructions

## Git Workflow

**IMPORTANT: Ne jamais faire de commits ou de push automatiquement.**

- Ne pas utiliser `git commit`
- Ne pas utiliser `git push`
- Laisser l'utilisateur gérer les commits et les déploiements lui-même

Quand des modifications sont faites, informer l'utilisateur des fichiers modifiés pour qu'il puisse commit et push manuellement.
