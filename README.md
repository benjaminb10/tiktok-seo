# Viewlify.app

> **Discover what makes TikTok videos go viral**

AI-powered TikTok analytics platform for creators, agencies, and brands. Analyze any account, detect viral hooks, and unlock engagement patterns that generate millions of views.

## 🚀 Features

- **Complete Analytics** - Detailed insights on views, engagement, likes, and comments
- **AI-Powered Insights** - Understand what makes videos viral with AI analysis
- **Competitor Tracking** - Monitor competitors and get real-time alerts
- **Viral Hook Detection** - Identify attention-grabbing patterns in top videos
- **Trend Spotting** - Discover trending sounds, hashtags, and formats early
- **Export & Reports** - CSV exports and automated reporting

## 🛠️ Tech Stack

- **Frontend**: React 19 + TanStack Start + TanStack Router
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **Backend**: Cloudflare Workers
- **Database**: D1 (SQLite) + Drizzle ORM
- **Analytics**: TanStack Table + Recharts

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🌐 Environment

The app runs on Cloudflare Pages with Workers.
